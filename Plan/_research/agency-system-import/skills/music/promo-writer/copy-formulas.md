## Hook Formulas

Every post needs a hook — the first line that stops the scroll. Pick the formula that fits the content.

### 1. Question Hook
Pose a question the album or track answers.

```
What happens when [premise]?
Ever wonder what [subject] sounds like as [genre]?
What if [concept] had a soundtrack?
```

**Best for**: Concept albums, story-driven tracks, thought-provoking themes
**Platforms**: Twitter (punchy questions), Instagram (caption openers), Facebook (comment bait)

### 2. Story Hook
Open with a narrative fragment that demands the rest.

```
It started with [inciting incident].
In [year], [subject] did something no one expected.
This track is about the moment when [turning point].
```

**Best for**: True-story/documentary albums, personal tracks, tracks with strong narratives
**Platforms**: Twitter threads, Instagram long captions, Facebook, YouTube descriptions

### 3. Contrast Hook
Juxtapose two unexpected elements.

```
[Genre A] meets [Genre B]. [Result].
The story of [dark topic] — told through [unexpected genre].
[Beautiful thing] built on [ugly truth].
```

**Best for**: Genre-blending albums, unexpected sonic choices, dark/light thematic contrast
**Platforms**: Twitter (concise contrast), TikTok (visual + audio contrast)

### 4. Number Hook
Lead with a specific, concrete detail.

```
[Number] tracks. [Number] true stories. One album.
[Number] months of research. [Number] sources. Every lyric verified.
[Number] tracks about [topic] you've never heard before.
```

**Best for**: Album announcements, documentary albums (emphasize research depth), tracklist reveals
**Platforms**: Twitter, Instagram carousels, TikTok text overlays

### 5. Quote Hook
Lead with the most quotable lyric or line from the track.

```
"[Quotable lyric]"

Track [N] from [album] — streaming now.
```

**Best for**: Tracks with strong individual lines, ongoing promotion, lyric-driven content
**Platforms**: Twitter (standalone quotes), Instagram (lyric graphics), TikTok (text overlay on audio)

### 6. Behind-the-Scenes Hook
Reveal something about the creative process.

```
This track almost didn't make the album. Here's why it did.
The hardest lyric to write on this album was [track] — here's the story.
I rewrote [track] [number] times before it felt right.
```

**Best for**: Post-release content, building artist authenticity, process-focused audiences
**Platforms**: Twitter threads, TikTok (talking to camera), Instagram Stories

## CTA Templates by Goal

### Drive Streams
```
Listen now: [link]
Stream [album name]: [link]
Full album on all platforms — link in bio.
Out now on [platform]. Hit play.
```

### Build Audience
```
Follow for more from this album.
Turn on notifications — more tracks dropping this week.
What do you want to hear next? Reply with your pick.
Save this post for when you need [mood the album creates].
```

### Drive Engagement
```
Which track hits hardest? Drop your answer below.
Rate this album 1-10 in the comments.
Tag someone who needs to hear this.
[Emoji] if this resonates. [Different emoji] if it doesn't.
What's your interpretation of track [N]?
```

## Post Structure Skeletons

### Twitter — Single Track Promo
```
[Hook — 1 line, under 100 chars]

[1-2 lines of context about the track]

[Streaming link]

#DiscoveryTag #GenreTag
```

### Twitter — Album Release Thread
```
Tweet 1:
[Album hook + cover art]

[Album name] is out now.

[Streaming link]

Tweet 2:
Here's what it's about:

[2-3 line concept summary]

Tweet 3-N:
Track [N]: "[Track Name]"
[1 line concept]
[Optional: quotable lyric]

Final tweet:
Full album: [link]

Thanks for listening. Which track hit hardest?

#NewMusic #GenreTag
```

### Instagram — Release Post
```
[Hook — first 125 chars are the preview]

[2-4 lines: album story, concept, personal angle]

[CTA: "Link in bio", "Save this", engagement question]

.
.
.
[15-20 hashtags in separate block]
```

### TikTok — Track Promo
```
[Under 150 chars — punchy, casual]

[1 discovery hashtag] [1 genre hashtag] [1 trending tag if applicable]
```
Video content does the heavy lifting — caption is secondary.

### Facebook — Release Announcement
```
[Hook — first 2-3 lines visible before "See more"]

[3-5 paragraphs: album story, track highlights, personal reflection]

[CTA: question to drive comments]

[Streaming link]

#Genre #NewMusic #ArtistName
```

### YouTube — Video Description
```
[Track/Album Name] - [Artist Name] (Official Audio/Lyric Video)

[1-2 line hook — visible in collapsed view]

Stream everywhere: [link]

---

About this track/album:
[2-3 paragraphs of context]

Credits:
[Production credits]

Follow [Artist]:
[Social links]

#ArtistName #AlbumName #GenreTag
```

## Hashtag Recipes

### By Campaign Phase

| Phase | Twitter | Instagram | TikTok |
|-------|---------|-----------|--------|
| Pre-release | #NewMusic + genre | #NewMusic + genre + #ComingSoon | #NewMusic + #MusicTok |
| Release day | #NewMusic + genre | #OutNow + genre + niche mix | #NewMusic + genre + trending |
| Post-release | #NowPlaying + genre | Genre + mood + niche | #NowPlaying + genre |
| Evergreen | Genre only | Genre + mood + niche | Genre + trending |

|-------|-------------|----------------|

### AI Music Tags (Use Sparingly)
- #AIMusic — general, recognizable
- #SunoAI — platform-specific
- #ClaudeCode — tool-specific

Don't put AI tags on every post. Lead with the music. Mention AI when it adds to the story.

## Engagement Post Ideas Beyond Announcements

Don't only post when you have something to sell. Between releases:

1. **"What I'm listening to"** — share other artists' music you enjoy
2. **"Making of" process shots** — DAW screenshots, lyric drafts, revision comparisons
3. **Polls** — "What genre should I try next?" or "Album art A or B?"
4. **Throwback** — revisit an older track with new context
5. **Fan interaction** — reshare someone who posted about your music
6. **Hot take** — opinion on music, genre, production (authentic, not inflammatory)
7. **Milestone celebration** — "500 streams!" (small wins are relatable)
8. **Question of the day** — "What song changed your life?"
9. **Lyric card** — one quotable line from a track, designed as a graphic
10. **"Did you know?"** — fact from your research (documentary albums)

## Tone Adaptation Guide

The same core message, adapted for each platform:

### Example: Album release announcement

**Core message**: New album "[Album Name]" is out — it's about [concept].

**Twitter** (punchy, direct):
```
[Album Name] is out now.

[Concept in one sentence].

[Number] tracks. Stream everywhere: [link]

#NewMusic #Genre
```

**Instagram** (visual + story):
```
[Album Name] is here.

[2-3 sentences about the story behind it — personal angle]

[CTA: "Link in bio. Which track hits hardest?"]

.
.
.
[Hashtag block]
```

**TikTok** (casual, authentic):
```
new album just dropped. [concept in casual language]
#NewMusic #Genre
```
(Video does the work — show your face, play a snippet)

**Facebook** (storytelling):
```
[Personal story about why you made this album — 3-5 paragraphs]

[Track highlights]

[CTA: "What do you think? I'd love to hear your take."]

[Link]
```

**YouTube** (informative):
```
[Album Name] - [Artist] (Full Album)

[Concept summary] | Stream everywhere: [link]

[Full credits, track listing, social links below]
```

The key: same message, native format. Never cross-post identical text.

## Language Adaptation

When writing in a non-English language or bilingual mode, adapt the copy naturally — don't just translate English templates word-for-word.

### Bilingual Post Format

Primary language first, divider, secondary language second. Hashtags only once (in English).

**Example: Facebook release post (DE + EN)**
```
[Album Name] ist ab sofort verfügbar!

[2-3 Absätze auf Deutsch — Konzept, Themen, was Hörer erwartet]

Jetzt streamen: [Streaming Link]

---

[Album Name] is out now!

[2-3 paragraphs in English — same content, not a literal translation]

Stream here: [Streaming Link]

#NewMusic #[Genre] #NewAlbum
```

### Language Rules

- **Hashtags**: Always in English for international discovery, regardless of copy language
- **Quoted lyrics**: Keep in original language. Add a brief translation in parentheses if the copy language differs
  - Example: *"Juste tourner, tourner, tourner"* (Just spinning, spinning, spinning)
- **Platform notes**: Always in English for consistency
- **Bilingual mode**: Write each version natively — adapt tone and idioms, don't translate literally
- **Twitter exception**: Bilingual mode does NOT stack two languages into one tweet — 280 chars is too tight. Instead, post each language as a separate tweet (or thread: Tweet 1 in primary language, Tweet 2 in secondary). All other platforms use stacked format.
