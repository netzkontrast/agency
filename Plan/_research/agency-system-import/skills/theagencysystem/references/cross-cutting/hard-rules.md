# Hard Rules — Cross-Cutting (privacy + craft guardrails)

Non-negotiables. A single violation of a NO-LABELS or purity rule = reject.

## NO-LABELS (privacy by design)

- **Never print an alter's personal name** in any music, lyric, design, or
  promo output. Use **function / role** only (host, rationalist, protector,
  caregiver, integrator, fighter, child_freeze, ambivalent,
  sexualized_override, collapsed, witness).
- **Never mark a voice switch** with a header, label, or announcement — the
  reader/listener recognizes the part by **syntax and vocabulary alone**. This
  is the one hard rule that is never violated.
- **No name-adlibs, no `[Section]`/character tags in lyrics.** Suno metatags may
  use descriptive form (`[male mid-baritone, weary, dry close-mic]`) but
  **never** the character form (`[Container]`).
- **Dissociation is amnesia-terror, never a crew menu.**
- **Whoever softens a threshold lies about the trauma.**

## ASDLS hard constraints

- **Aesthetic ban list (purity check):** no 1980s/90s retro-futurism — no
  synthwave, outrun, purple-orange gradients, neon 80s grid, VHS nostalgia,
  Blade-Runner pastiche.
- **No analog artifacts in the digital layer:** no paper grain, brush duktus,
  pencil smudging, watercolor, visible paper texture, organic curves.
- **No daylight, no sun, no natural elements.** No cute, no soft lighting.
- **Gradient ban:** state colors meet as hard edges / blocky pixel-shifts only.
- **5% rule:** ≤5% state color over ≥95% Terminal Black / Deep Charcoal.
- **One core symbol per image; one tier per image** (mixing tiers forbidden).

## Negative-prompt list (forbidden)

```
--no 1980s retro, synthwave, outrun, purple-orange gradient, neon grid,
daylight, sun, natural elements, cute, soft lighting, watercolor, analog
painting, visible paper texture, lens flare, organic curves
```

## Prioritization matrix (resolve conflicts in order)

1. Emotional system-state (highest — even at the cost of legibility)
2. Clinical syntax & materiality
3. Semiotic function & motif
4. Visual cache (lowest — random generative artifacts)
