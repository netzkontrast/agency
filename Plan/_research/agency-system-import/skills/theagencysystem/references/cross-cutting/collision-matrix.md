# Collision Matrix — Cross-Cutting

Function-keyed phobia/collision network (merges the music phobia network +
the novel collision matrix). **Function names only — never personal names.**
`A → B` = A fears/clashes with B. `A ↔ B` = mutual.

**Classes:** ANP = {host, rationalist}. EP = {protector, caregiver, fighter,
child_freeze, ambivalent, sexualized_override, collapsed}. Meta =
{integrator (ISH), witness}.
