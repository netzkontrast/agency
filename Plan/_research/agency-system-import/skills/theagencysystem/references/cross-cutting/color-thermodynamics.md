# Color Thermodynamics — Cross-Cutting

Color is a **functional variable bound to a psychometric axis**, never
decoration. Base = dark-mode void; state color is sparse and stinging.

## The yellow typologies (analog light-sources / metaphysical states)

| Yellow | State | Behavior |
|---|---|---|
| **Latency Violet** `#3B3355` | Dissociation, dwindling bandwidth, derealization | Vanishes into dark, forms no clear edge (the cold start of the arc) |
| **Signal Yellow** `#FFD700` | Acute danger, alarm, panic | Glaring neon line cutting Void Black; tears the retina |
| **Flame / Corrupted Yellow** | Destructive collapse / toxic data decay | Corrupted Yellow `#8B8B00` infects + dulls neighbors; Flame Orange `#FF4500` overheats |
| **Kintsugi Gold** `#FFDF00` | Repair, resynthesis, insight-breakthrough | Precise luminous PCB circuitry filling cracks, not organic gold |

## Hex-typology table (state colors — the 5% rule)

95% background paradigm, ≤5% one state color. Hard edges only — gradient ban.

| Color | Hex | Signals |
|---|---|---|
| **Terminal Black** | `#0B0D17` | Default operative background; isolation, infinite mainframe depth |
| **Deep Charcoal / Latency Gray** | `#1A1D24` | Inactivity, latency, infrastructure standby; brutalist concrete |
| **System Blue** (Control) | `#003366` | Absolute control, cold logic, firewall; stability + emotional coldness |
| **Corrupted Yellow** | `#8B8B00` | Trauma, freeze, resignation, toxic decay; *infectious* (desaturates neighbors) |
| **Signal Yellow** (Alert) | `#FFD700` | Acute danger, intrusion detection, panic |
| **Terminal Green** | `#00FF9F` | Uncoded noise, operational unrest, raw data streams |
| **Flame Orange** | `#FF4500` | Destructive collapse, rage, kernel overload, thermal breach |
| **Clean Ping** (Hope) | `#FCEE0C` | Fragile undamaged stream, innocence; tiny pixel/hairline only |
| **Latency Violet** | `#3B3355` | Dissociation, identity loss, derealization |
| **Kintsugi Gold** | `#FFDF00` | Repair algorithm, integration, breakthrough |

**Law of exclusion:** Flame Orange and Clean Ping never share a frame unless
one overwrites the other as a sharp glitch.
