# Glitch Typology — Cross-Cutting

Glitches are **diagnostic, never decorative**. Each names a system-state and
maps to a digital prompt term + an analog stroke-grammar. *The machine speaks
in vectors, but it bleeds in ink.*

| Glitch | State / affect | Digital (ASDLS) | Analog hand |
|---|---|---|---|
| **Packet loss** | Latency, derealization, dissociative amnesia, "not-really-here" | Contours break off mid-line, vanish into noise, low-opacity wireframes | Lines break for no reason; estompe/diluted wash bleeds object into nothing |
| **Chromatic aberration** (RGB-split) | Cognitive dissonance, split memory access, phase mismatch | Sharp red/blue channel shift at edges only | Black fineliner motif; contours offset L/R in red + blue/cyan pen |
| **Datamoshing / macro-blocking** | Traumatic memory leak, repression bleeding into present | Hard rectangular pixel-shifts, corrupted JPEG/codec failure | Graphite/charcoal lifted in rectangular blocks, smeared wrong direction |
| **Vector jitter** | Systemic overload, panic, acute hyperarousal | Overlapping faulty paths, aggressive dense hatching | Trembling, lifted-and-restarted lines; pressure scores/incises the paper |
| **Scanline tear** | Flickering instability, unstable identity | Horizontal CRT-style line breakaway | Doubled contours (gray + magenta + green, never aligned) tearing horizontally |
| **Whiteout / erased vectors** ("Format C:") | Deletion logic, catatonic shutdown | Hard inexplicable cut-offs, rigid white rectangles | White-out / scalpel cuts; emotional line ends at a straight edge in nothing |

## Escalation (state-machine, never blend tiers)

- **T0 Homeostasis** — no glitches; flawless vectors.
- **T1 Latency/Freeze** — packet loss, fading opacity, ghost contours.
- **T2 Alert** — *light* chromatic aberration at edges + densified hatching.
- **T3 Kernel Panic** — extreme datamoshing, torn geometry, grid destruction.
- **T4 Safe Mode** — scars visible as dead furrows, no longer glowing.
