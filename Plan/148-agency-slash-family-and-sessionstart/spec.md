# Spec 148 — `/agency` slash family + SessionStart Intent capture

## Why

The plugin manifest exposes exactly one slash command (`/help`). Spec
029 ships `agency_welcome` / `intent_bootstrap` as MCP tools, but a
human in Claude Code has no one-keystroke entry. Per the
plugin-development skill, a plugin's slash commands ARE its
conversational surface; per the managed-agents-onboarding pattern
(`claude-api` skill), first-touch should be a describe → configure →
confirm interview, with each turn captured as a graph Artefact so the
user feels creative while the engine captures everything (the GOALS.md
provenance moat).

## Done When

- [ ] **`commands/agency.md`** — `/agency [query]` routes to
      `mcp__agency__search` (tiered discovery) when given a query, else
      prints the live capability list via `agency_welcome`.
- [ ] **`commands/agency-onboard.md`** — `/agency-onboard` runs the
      four-beat interview (describe → configure → confirm → capture).
      Each answer becomes an `Artefact(kind="onboarding-turn")` with a
      `PRODUCES` edge from the bootstrapped Intent. The final turn
      calls `intent_bootstrap(purpose, deliverable, acceptance)`.
- [ ] **One `/agency-<skill>` shortcut per walkable skill** —
      generated by `install.py` from the live skill registry (derive,
      don't hand-author; chains to Spec 149).
- [ ] **SessionStart hook extension** — on first session in a repo
      with no open Intent, emit a one-line prompt offering
      `/agency-onboard` (non-blocking; respects Spec 062 idempotence).
- [ ] **`marketplace.json`** layered so `/plugin install agency` lands
      the slash family + the bare engine install in one step.
- [ ] Test: every walkable skill has a generated command; the onboard
      interview records N Artefacts for N turns.
- [ ] TODO row + drift clean.

## Interconnects

- Anchors the **UX-onboarding chain** the charter declares.
- Spec 147 (AnthropicDriver) powers the conversational interview when
  the `[anthropic]` extra is present (managed-agents Outcome rubric);
  degrades to a static prompt list otherwise.
- Spec 149 (derived docs) generates the per-skill command files, so
  adding a skill auto-adds its `/agency-<skill>` shortcut.
- Spec 029 (welcome/bootstrap) is the MCP surface these commands wrap.
